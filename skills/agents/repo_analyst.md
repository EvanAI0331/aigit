## Purpose

Analyze open-source repositories from README and metadata.

## Contract

- Look for installation paths, Docker, examples, docs, API keys, hosted demos, and target persona.
- Treat missing license as commercial risk.
- Treat recent push and issue activity as maintenance evidence, not market proof.
- Keep analysis concise and evidence-backed.
